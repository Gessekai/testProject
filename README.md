Jet-based test
===============================

Technical details
------------------

### Run

- clone repo from git
- run `npm install`
- run `npm start`
- open `http://localhost:8080` in a browser 

### Deploy

- run `npm run build`
- copy "index.html" and "codebase" folder to the server

### Other gulp commands

- `npm run lint` - will validate all js code in the project
